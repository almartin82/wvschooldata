# Process district-level enrollment data

Combines FTE grade data with headcount totals.

## Usage

``` r
process_district_enr(fte_data, headcount_data, end_year)
```

## Arguments

- fte_data:

  Data frame with FTE enrollment by grade

- headcount_data:

  Data frame with headcount enrollment

- end_year:

  School year end

## Value

Processed district data frame
